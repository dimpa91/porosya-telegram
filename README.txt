== Description ==

Porosya Telegram helps send wp notifications to telegram. Any text to any telegram.

= Features: =

* Send messages to telegram
* Check status of your bot

== Installation ==

1. Upload `porosya-telegram` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Settings -> KJM Admin Notices' and activate both 'KJM Admin Notices' and 'Notice Post Type'.
1. Start creating your notices by going to 'Manage KJM Admin Notices' and have fun!
